# USR-Connect ⚽💜

Application de gestion des bénévoles pour le tournoi **Solida'Foot** de l'US Reventin.

## Fonctionnalités
- ✅ Visualisation du planning des missions.
- ✅ Inscription et désinscription en un clic.
- ✅ Affichage de la liste des bénévoles par créneau.
- 🚧 Création de nouveaux créneaux (En cours).

## Installation
Projet basé sur Laravel 11, Inertia.js et Vue 3.
